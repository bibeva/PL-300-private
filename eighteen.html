Question 180

You are developing a Power BI report for a real estate company that tracks property sales performance. The report includes a visual that displays the profit margin for each property sold. The profit margin is calculated using a DAX measure.
The real estate agents have requested that the report adheres to specific formatting guidelines for clarity:
- Profit margins must be displayed with two decimal places for consistency (e.g., 12.34%).
- Negative profit margins, indicating a loss, must be displayed in red font with parentheses for easy identification (e.g., (5.67%)).
What two actions should you perform in Power BI to meet these formatting requirements? Each correct answer presents part of the solution.

  • A. For the visual, apply conditional formatting to the background color of the data points.
  • B. Configure the DAX measure to use a custom format string.
  • C. For the visual, apply conditional formatting to the font size of the data labels.
  • D. For the visual, set "Value decimal places" to 2 in the formatting options
  • A. For the visual, apply conditional formatting to the background color of the data points.
  • B. Configure the DAX measure to use a custom format string.
  • C. For the visual, apply conditional formatting to the font size of the data labels.
  • D. For the visual, set "Value decimal places" to 2 in the formatting options

Explanation: To display profit margins with two decimal places and format negative values with parentheses and red font, you should configure the DAX measure with a custom format string that specifies percentage formatting with two decimals and negative number style. Additionally, you should set "Value decimal places" to 2 in the visual’s formatting options to ensure the displayed values consistently show two decimal places. Conditional formatting for background or font size is not needed for this requirement.
