Question 160

After deploying a detailed report to a Power BI dashboard, feedback indicates a strong user interest in querying the data through natural language. What is the correct method to enable natural language queries directly from the dashboard?
Solution
Implement a dedicated Q&A button on the dashboard, enabling users to initiate queries with ease. Does this solution meet the goal?

  • A. Yes
  • B. No
  • A. Yes
  • B. No

Explanation: Power BI’s Q&A feature allows users to ask natural language questions about the data directly from a dashboard. Adding a dedicated Q&A button provides an intuitive way for users to interact with the dataset using natural language queries, meeting the goal of enabling easy, self-service data exploration.
