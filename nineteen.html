Question 190

Imagine you're a data analyst working for a telecommunications company. You're analyzing customer data to understand usage patterns and preferences for different mobile phone plans. Your dataset includes a column called "MonthlyUsage" that records the number of minutes each customer uses per month.You want to segment customers into different usage groups (low, medium, high) to tailor marketing campaigns and plan service offerings.
Question: Which PowerQuery feature can you use to categorize customers into the following usage groups based on their "MonthlyUsage":
- Low Usage: 0-100 minutes
- Medium Usage: 101-500 minutes
- High Usage: 501+ minutes

  • A. Create a new table with the usage group definitions and establish a relationship with the customer data table.
  • B. Use the "Conditional Column" feature and create a set of if statements to categorize each event.
  • C. Use the "Enter Data" feature and manually write each customer with the correct MonthlyUsage categorization
  • A. Create a new table with the usage group definitions and establish a relationship with the customer data table.
  • B. Use the "Conditional Column" feature and create a set of if statements to categorize each event.
  • C. Use the "Enter Data" feature and manually write each customer with the correct MonthlyUsage categorization

Explanation: The Conditional Column feature in Power Query allows you to define rules based on existing column values, such as "MonthlyUsage," to create new categorical columns like Low, Medium, and High usage. This method is efficient, dynamic, and automatically applies to all rows in the dataset, unlike manually entering data or creating separate tables.
