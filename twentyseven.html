Question 270

Scenario: In Power BI Desktop, you're working with a dataset that includes an inventory table of art pieces displayed on a report page. The table includes columns for 'Art Title' and 'Art Image URL,' which contains links to images of the artworks hosted online.
Objective: You need to configure the table visual to display the art images directly, instead of showing the URLs.
What adjustment should you make to the 'Art Image URL' column in the table visual to display the images directly within the table?

  • A. From the Visual Formatting menu, choose Values and enable 'Display URLs as Icons'.
  • B. Set the Data Category for the 'Art Image URL' field to 'Web URL'.
  • C. Change the Data Type of the 'Art Image URL' field to 'Binary'.
  • D. Assign the Data Category of the 'Art Image URL' field to 'Image URL'.
  • A. From the Visual Formatting menu, choose Values and enable 'Display URLs as Icons'.
  • B. Set the Data Category for the 'Art Image URL' field to 'Web URL'.
  • C. Change the Data Type of the 'Art Image URL' field to 'Binary'.
  • D. Assign the Data Category of the 'Art Image URL' field to 'Image URL'.

Explanation: In Power BI Desktop, to display images in a table or matrix visual instead of showing plain URLs, you must set the Data Category of the column containing the image links to Image URL. This tells Power BI that the field contains image addresses, enabling the visual to render the actual images. Setting it as a generic Web URL or changing the data type to binary will not display the images directly.
