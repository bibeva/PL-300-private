Question 210

You are tasked with building a report that highlights your company's most valuable customers. This report should showcase the top 20 customers based on their total purchase value over the last year.
You have a dataset with customer details, purchase history, and order values. Which filter type and level should you use to achieve this?
Filter Type: [Answer Choice]
Level: [Answer Choice]

  • A. Advanced filter, Page level
  • B. Relative date filter, Report level
  • C. Top N filter, Visual level
  • D. Basic filter, Page level
  • A. Advanced filter, Page level
  • B. Relative date filter, Report level
  • C. Top N filter, Visual level
  • D. Basic filter, Page level

Explanation: A Top N filter allows you to display only the top 20 customers based on a measure like total purchase value. Applying it at the visual level ensures that the ranking affects only the specific visual highlighting these customers, without altering other visuals or pages in the report.
