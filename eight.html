Question 80

You are tasked with analyzing the distribution of character lengths within a text column named 'Feedback' in a Power BI Desktop project. The goal is to broadly understand the distribution while minimizing impact on the model size. How should you approach this task to analyze the character length distribution of the 'Feedback' column most efficiently?

  • A. Create a DAX measure that calculates the length of 'Feedback' entries on-the-fly for analysis in visuals.
  • B. Create a DAX calculated column to determine the length of each 'Feedback' entry.
  • C. Use the Column Profiling Tools in Power Query Editor to examine the 'Feedback' column's length distribution without adding any columns to your model.
  • A. Create a DAX measure that calculates the length of 'Feedback' entries on-the-fly for analysis in visuals.
  • B. Create a DAX calculated column to determine the length of each 'Feedback' entry.
  • C. Use the Column Profiling Tools in Power Query Editor to examine the 'Feedback' column's length distribution without adding any columns to your model.

Explanation: Column profiling in Power Query allows you to inspect the distribution of data, including value lengths, without creating additional calculated columns or measures that increase the dataset size. This approach efficiently provides insights into character lengths while keeping the model lean, unlike calculated columns or on-the-fly DAX measures, which would consume memory and impact performance.
