Question 220

A business intelligence (BI) specialist has configured a Power BI dataflow to pull data using DirectQuery from an on-site Microsoft SQL Server. This dataflow also has the Enhanced Dataflows Compute Engine enabled. Your task is to incorporate this dataflow into a report while adhering to the following objectives:
- Reduce reliance on real-time online data processing.
- Decrease both the calculation and display times of report visuals.
- Ensure the data includes records from the current year until the day before the present.
What action should you take?

  • A. Establish a connection to the dataflow using DirectQuery.
  • B. Establish a connection to the dataflow using DirectQuery and set up a gateway for this dataset.
  • C. Establish a connection to the dataflow using Import mode and arrange for daily updates.
  • D. Establish a connection to the dataflow using Import mode and configure an hourly data refresh via Microsoft Power Automate.
  • A. Establish a connection to the dataflow using DirectQuery.
  • B. Establish a connection to the dataflow using DirectQuery and set up a gateway for this dataset.
  • C. Establish a connection to the dataflow using Import mode and arrange for daily updates.
  • D. Establish a connection to the dataflow using Import mode and configure an hourly data refresh via Microsoft Power Automate.

Explanation: Using Import mode loads the data into Power BI’s in-memory engine, which reduces reliance on real-time queries and improves visual calculation and display performance. Scheduling daily refreshes ensures the data remains current (e.g., including records up to yesterday) without the performance overhead of DirectQuery.
