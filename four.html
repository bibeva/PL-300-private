Question 40

In Power BI, you're tasked with assessing the quality of your dataset after using Power Query for data inspection. Determine the veracity of each statement below. If it’s accurate, choose Yes. If not, choose No.
- 'Column profile' in Data Preview is by default limited to evaluating the data pattern within a subset of 1,000 rows.
- Utilizing 'Column quality' in Data Preview allows for a complete assessment across all data points within the dataset.
- Activating continuous data update settings in Power Query is essential for immediate reflection of changes in the data preview.

  • A. Yes, Yes, No
  • B. Yes, No, No
  • C. Yes, Yes, Yes
  • D. No, Yes, No
  • A. Yes, Yes, No
  • B. Yes, No, No
  • C. Yes, Yes, Yes
  • D. No, Yes, No

Explanation: Column profile is limited to the first 1,000 rows by default, so that is correct. Column quality also uses only the preview subset unless you enable "Column profiling based on entire data," so it does not always assess the full dataset. Continuous data update is not required for preview changes, so the last statement is false.
