Select whether the following statement is true or false:
"Bins in Power BI can be used to categorize continuous data fields into discrete intervals, but they cannot be applied to text data."

  • A. True
  • B. False
  • A. True
  • B. False

Explanation: In Power BI, bins are designed to group continuous numeric data into discrete intervals, allowing for easier analysis of distributions or ranges. Text fields, being categorical rather than continuous, cannot be directly binned using this feature.
