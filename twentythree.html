Question 230

You have a clustered column chart in Power BI that displays a measure called SalesAmount as the value and a field named Salesperson on the axis. The SalesAmount measure represents the sales revenue generated by each salesperson in US dollars.
You need to add a reference line to the chart that indicates which salespeople are above the median sales amount.
Solution: You create a median line using the SalesAmount measure.
Does this solution meet the goal?

  • A. Yes
  • B. No
  • A. Yes
  • B. No

Explanation: Adding a median line based on the SalesAmount measure will display the overall median across all sales, but it does not automatically indicate which salespeople are above the median. To meet the goal, you would need to create a measure that calculates the median per salesperson and then use conditional formatting or a reference line to highlight salespeople exceeding that median.
